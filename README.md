

# 🤖 DEBUG HERO — Cyber-Labyrinth

![Status](https://img.shields.io/badge/STATUS-DEVELOPMENT-00ff41?style=for-the-badge&logo=react)
![Engine](https://img.shields.io/badge/ENGINE-VITE-646CFF?style=for-the-badge&logo=vite)
![Style](https://img.shields.io/badge/CYBERPUNK-black?style=for-the-badge)

**Debug Hero** — це освітня логічна гра, створена для того, щоб перетворити виснажливий пошук помилок у коді на захопливу кіберпанк-пригоду. Ви граєте за дрона-налагоджувача, який має врятувати систему від вірусу **ChaosCompiler**.

---

## 🕹️ Геймплейні механіки

* **Класичний Платформер:** Керуйте героєм (🤖), долаючи перешкоди в різних цифрових світах.
* **Термінал Бою:** Зустріч із багами активує інтерактивний термінал.
* **Типи викликів:**
    * **Logic Fix:** Виправлення критичних помилок безпосередньо у редакторі коду.
    * **Drag-and-drop:** Розстановка блоків алгоритму в правильному порядку.
* **Візуальні ефекти:** Переможені баги розсипаються на бінарні частинки (01101).

## 🛠️ Технологічний стек

* **Core:** React.js (Vite)
* **State Management:** React Hooks (`useState`, `useRef`, `useCallback`)
* **Physics:** Custom Gravity & Collision Engine
* **Styling:** Pure CSS (Industrial/Archive aesthetic)

## 👨‍💻 Розробка

Проєкт використовує мінімалістичну збірку Vite з підтримкою HMR (Hot Module Replacement).

- **Створення рівнів:** Всі рівні конфігуруються у файлі `src/data/gameConfig.js`.
- **Компоненти:** Логіка завдань ізольована у `TaskHandler.jsx` для легкої масштабованості.
